# Detec-o_Explic-vel_e_Respons-vel_de_Toxicidade_Portugu-s
Projeto de detecção explicável e responsável de toxicidade em português brasileiro, com avaliação entre diferentes conjuntos de dados, uso de atributos lexicais interpretáveis, embeddings e modelos de aprendizado de máquina para classificação de textos tóxicos e não tóxicos.
